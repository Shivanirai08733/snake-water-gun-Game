# snake-water-gun-Game
## Snake Water Gun Game in Python

A simple yet interactive **Snake Water Gun** game built in **Python** using the **random** module and **conditional statements**. The program allows the user to play against the computer, where the computer randomly selects one of the three options: **Snake**, **Water**, or **Gun**. The user enters their choice, and the program compares both selections using conditional logic to determine the winner based on the game rules.

The project demonstrates the practical use of Python fundamentals, including user input handling, random number generation, conditional (`if-elif-else`) statements, variables, and basic program flow. It also includes input validation to ensure that the user enters a valid choice before the game proceeds.

This beginner-friendly project is an excellent way to understand decision-making in Python while creating a fun command-line game. It can be further enhanced by adding features such as multiple rounds, score tracking, replay options, difficulty levels, or a graphical user interface (GUI).
